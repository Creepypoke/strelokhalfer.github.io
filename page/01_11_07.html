Если вы экспортируете проект на Windows, вы не сможете распростронять только Game.exe, вам придётся распростронять вместе со всей папкой и вложенными файлами.

Но используя Enigma Virtual Box мы можем запаковать все данные в один исполняемый файл.

Качаем и устанавливаем

  1. Скачайте [Enigma Virtual Box] по приведённой ссылке ниже.
    http://enigmaprotector.com/en/downloads.html
  2. Запустите "enigmavb.exe" и установитье туда, куда вам удобно.

Как использовать

  1. Запустите Enigma Virtual Box.
    П/п. Если у вас интерфейс на русском, советую поменять его на английский.
  2. Нажмите [Browse...] рядом с полем "Enter Input File Name:", и выберите исполняемый файл в папке вашего проекта. Поле "Enter Output File Name:" заполнится автоматически(Можете поменять место и название, если захотите.)
  3. Нажмите [Add], выберите [Add Folder Recursive].
  4. Выберите папку проекта, в следующем окне нажмите [OK].

    ↓
  5. Убедитесь, что в окне по центру добавилась ваша папка, нажмите [Process].
  6. Созданный файл будет лежать в папке, указанной в "Enter Output File Name:".
